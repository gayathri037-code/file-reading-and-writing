📁 File Read and Write Operations using fs Module (Node.js)

📌 Aim

To perform file read and write operations using Node.js fs module.

🧠 Description

In Node.js, the fs (File System) module is used to interact with files.
This project demonstrates:

- Writing data into a file
- Reading data from the file

⚙️ Requirements

- installed

🚀 Steps to Run

1. Install Node.js
2. Open terminal in project folder
3. Run the program:
   node file.js

💻 Code Explanation

- "fs.writeFile()" → Creates a file and writes data
- "fs.readFile()" → Reads data from file

📂 Files Included

- "file.js"
- "sample.txt" (auto generated)

▶️ Output

File written successfully
File content: Hello! This is file write operation.

🎯 Result

Successfully performed file read and write operations using Node.js.

📌 Conclusion

The fs module helps in handling file operations easily in Node.js applications.
